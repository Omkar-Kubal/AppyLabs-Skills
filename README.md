# AppyLabs Skills

Curated AI agent skills for mobile app development workflows. Each skill is a structured markdown document loaded by an AI agent at runtime to guide specialized tasks.

## Skills

| Skill | Description |
|---|---|
| **android-mobile-design** | Android UI/UX design guidance using Material Design 3 and official Android docs |
| **app-market-research** | 10-phase competitive intelligence and product strategy for mobile apps |
| **mobile-app-playbook** | End-to-end Android app development guide (MVP to Play Store launch) |
| **mobile-mockup-imagegen** | Generate high-fidelity mobile app UI screen mockups from design specs |
| **saas-to-android** | Reverse-engineer any SaaS product into a full Android app execution plan |

## Usage

Add this repository to your AI agent's skills directory. The agent will load the appropriate skill when a matching user query is detected.

```sh
git clone https://github.com/Omkar-Kubal/AppyLabs-Skills.git
```

Targeted at Indian mobile app developers and founders, with region-specific pricing, compliance, and market data.
